Estilos con CSS

Hojas de Estilo en Cascada

Se utilizan distintas formas segun las necesidades que se tengan con los estilos.

Existen 3 formas para incorporar los estilos:

1. Estilo en Linea

Se coloca el atributo style a la etiqueta, y como valor las propiedades con sus valores en css

Se utiliza para estilos especificos

Este es el ejemplo

2. Estilo Interno

La regla CSS se realiza en una etiqueta llamada style

Se utiliza para estilos mas generales dentro de una pagina, optimizando su carga sin traer un archivo extra

Este es el ejemplo

3. Hoja de Estilos Externa

Se crea un archivo con extension .css donde se colocan las reglas de estilo. Luego, se indexa al archivo HTML que va a aplicar los estilos.

Se utiliza para estilos generales, que se quieran compartir entre diferentes paginas

Para realizar esto, se coloca la etiqueta link:css dentro de la etiqueta head, y se le pasan 2 atributos:
